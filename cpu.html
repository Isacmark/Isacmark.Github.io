Cpu: Som även står för Central Processing Unit är vad man skulle kunna säga “datorns hjärna”. Processorn i datorns uppdrag i datorn är att utföra program och beräkningar. Processorer kan man hitta i de flesta maskiner nuförtiden, som i t.ex bilar. Det finns två varianter av en processor man kan köpa. CISC: Complex Instruction Set Computing. Den här processorn skapades för att utföra de svårare och längre uppgifterna fast lite långsammare än ett RISC Cpu. RISC: Också känt som på engelska, Reduced Instruction Set Computing och den är gjord för att utföra enkla uppgifter snabbt.
